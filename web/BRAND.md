# España Transparente · Identidad visual

## Concepto

El símbolo es una estructura descendente que concentra la atención en un punto final. Funciona como marca de un portal de datos: sobria, reconocible en tamaños pequeños y sin referencias partidistas o institucionales oficiales.

## Sistema

| Uso | Asset | Tamaño recomendado |
| --- | --- | --- |
| UI principal | `LogoMark` | 28-32 px en header |
| Hero / presentación | `LogoMark` | 72-96 px |
| Favicon | `public/favicon.svg` + `public/favicon.ico` | 16, 32, 48 px |
| Apple touch icon | `public/apple-touch-icon.png` | 180 x 180 px |
| Web app | `public/icons/icon-192.png`, `public/icons/icon-512.png`, `public/icons/maskable-512.png` | PWA estándar |
| Open Graph | `public/brand/og-image.png` | 1200 x 630 px |
| GitHub / README | `public/brand/github-banner.svg` | 1280 x 640 px |
| Marca aislada | `public/brand/espana-transparente-mark.svg` | SVG vectorial |

## Color

| Token | Valor | Uso |
| --- | --- | --- |
| Ink | `#1A1612` | texto principal, marca, superficies oscuras |
| Paper | `#FAFAF7` | fondo base y versiones inversas |
| Signal | `#A1242C` | foco, navegación activa, excepciones y puntos destacados |

## Reglas de uso

- Usar el símbolo completo en UI, favicons, PWA y redes.
- Usar la versión con ápice rojo cuando haga falta reforzar el punto de atención.
- Usar la versión blanca solo sobre fondos oscuros.
- No combinar el símbolo con escudos, banderas o siglas partidistas.
- Reservar `Signal` para estados activos, foco y señales puntuales; no usarlo como color decorativo masivo.
